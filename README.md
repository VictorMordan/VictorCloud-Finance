Victor Finance Cloud

Victor Finance Cloud es una aplicación de gestión financiera personal diseñada para ofrecer control total sobre los gastos mensuales en tiempo real. Construida con un enfoque en la simplicidad y la seguridad, permite a los usuarios sincronizar sus datos en la nube mediante autenticación oficial de Google.
Link del Proyecto: https://victormordan.github.io/VictorCloud-Finance/
Características Principales

Autenticación con Google: Acceso seguro sin necesidad de recordar nuevas contraseñas.

Sincronización Cloud: Tus datos disponibles en cualquier dispositivo gracias a Firebase Firestore.
